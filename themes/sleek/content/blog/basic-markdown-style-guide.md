## Inline formatting

You can use all sorts of inline formatting in markdown. You can write something in **bold**. And _italics_ too. How about ~~strikethrough~~? You got it. Don't forget about inline code, you got that too. `Hello world!`.

## Code Blocks

Syntax highlighting is done using [Prism.js](https://github.com/PrismJS/prism). You can customise to whichever theme you want from the [plenty available prism themes](https://github.com/PrismJS/prism-themes).

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <title>Example HTML5 Document</title>
  </head>
  <body>
    <p>Test</p>
  </body>
</html>
```
